FoosBall

This project is a FoosBall game implemented using Java wherein we used Swing API, most of the OOPS concepts and patterns. It was a group project done in a team of 4.

Technologies and Concepts Used

We used Swing API which is a GUI widget toolkit for Java. Swing uses AWT(Abstract Window Toolkit), the more traditional way of creating an operating system window to create GUI features. We used SVN to collaborate.

Code Structure

The game has multiple threads running. Each team has a thread of it's own. Along with that wee have a thread for the Ball logic. One thread is dedicated to render the GUI.
